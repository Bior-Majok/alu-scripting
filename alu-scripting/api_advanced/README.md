# API Advanced

This project introduces how to use the Reddit API using Python and the requests module.

## Files
- 0-subs.py — Returns number of subreddit subscribers.
- 1-top_ten.py — Prints top ten hot posts.
- 2-recurse.py — Recursively lists all hot titles.
- 100-count.py — Counts keyword occurrences in hot posts.

## Requirements
- Python3
- Requests module
- Custom User-Agent
