This project focuses on interacting with the Reddit API using Python.
You will learn:

- How to read API documentation
- How to use pagination
- How to parse JSON results
- How to make recursive API calls
- How to sort a dictionary by value

## Requirements
- Python 3 (Ubuntu 14.04)
- PEP8 code style
- Must use the `requests` module
- All files must be executable
- No redirects should be followed for invalid subreddits

## Files
### 0-subs.py
Contains the function `number_of_subscribers(subreddit)` which returns the
number of subscribers for a given subreddit. Returns 0 if the subreddit is
invalid.

### 0-main.py
Used to test the function with any subreddit.
