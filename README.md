# 1. Top Ten Reddit Posts

This project contains a Python function that queries the Reddit API and prints the titles of the first 10 hot posts for a given subreddit.

## Files

- `1-top_ten.py` – Contains the function `top_ten(subreddit)`.
- `1-main.py` – Script to test the `top_ten` function from the command line.

## Requirements

- Python 3.x
- `requests` library

pip install requests
