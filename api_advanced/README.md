# API Advanced

This project contains Python scripts that interact with the Reddit API to practice API calls, pagination, recursive functions, and data parsing.

## Learning Objectives
- Read API documentation to find endpoints
- Use an API with pagination
- Parse JSON results from an API
- Make recursive API calls
- Sort a dictionary by value

## Requirements
- Python 3.4.3
- PEP 8 style
- All files must be executable
- Use Requests module for HTTP requests
- Custom User-Agent required for Reddit API

## Files
- `0-subs.py` - Function that queries Reddit API and returns subscriber count for a subreddit

## Usage
```bash
python3 0-main.py programming
```
